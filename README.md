# Créa&Sport Shop — version propre Vercel

Projet React + Vite propre, prêt pour Vercel.

## Commandes

```bash
npm install
npm run dev
npm run build
```

## Déploiement Vercel

- Framework: Vite
- Build Command: npm run build
- Output Directory: dist
- Root Directory: ./

Le fichier `vercel.json` force aussi ces réglages.
